# KUWoC - Kathmandu University Winter Of Code

Official website repository for Kathmandu University Winter Of Code (KUWoC) 
- a 2-month long open-source program for students.

## About KUWoC
KUWoC (Kathmandu University Winter Of Code) is a 2-month long initiative to introduce students to the world of open source development. This repository contains the official website providing comprehensive information and guides for both contributors and organizations participating in the program.
